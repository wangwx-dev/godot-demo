# 末日搜打撤

俯视角 2D 丧尸生存 roguelike——把"吸血鬼幸存者"式的自动战斗成长，装进一场 **8 天求生的搜打撤（搜刮 / 战斗 / 撤离）** 循环里。

> 基于 Godot 4.7（标准版 / GDScript / GL Compatibility）。当前为抢先体验版 v0.9。

## 玩法一句话

无线电约定第 8 天来接应，但接应点已被尸巢占据。用 8 天做准备，最后打一场总攻杀出去——**每一天去哪张图，就是这一天怎么花**：搜物资、练构筑、还是回血续航。任何时候死亡 = 本局收益全部丢失。

### 三层循环
- **秒级**：移动走位躲丧尸，武器自动索敌开火（不用瞄准），翻滚保命。
- **分钟级（单张图）**：战争迷雾下探索，搜资源点攒物资/经验/货币，找到**载具**就是找到出口；图内有 5 分钟死线，拖太久压力垂直崩溃。
- **局级（一整局约 40 分钟）**：8 天线性推进，每天 2 个候选图（战斗 / 精英 / 休整 / 商店），选一辆载具出发就放弃其余。天数耗尽进总攻 Boss 战。

## 操作

| 键 | 动作 |
|---|---|
| **WASD / 方向键** | 移动（武器自动攻击，站位即瞄准） |
| **空格** | 翻滚（短冲刺 + 无敌帧，有冷却） |
| **E** | 交互（长按上载具出发 / 休整图服务 / 信号弹） |
| **ESC** | 暂停菜单（继续 / 设置 / 回主菜单 / 退出） |
| **F1** | 调试角标（FPS / 种子 / 状态，开发用） |

升级时弹三选一强化（暂停游戏，可花货币重随或跳过）；背包满了开箱会弹替换界面。

## 内容规模（v0.9）

- **12 把武器**：6 主武器（球棒 / 手枪 / 霰弹枪 / 钉枪 / 链锯 / 复合弓）+ 6 副武器（燃烧瓶 / 手雷 / 捕兽夹 / 闪光弹 / 诱饵收音机 / 肾上腺素），各带专属精通强化。
- **3 种丧尸** + **词缀精英**（狂暴 / 召唤 / 坚韧）。
- **6 个主题地图模块** 2×2 随机拼接，每局地图不同；休整 / 商店 / 总攻各有专属场景。
- 白 / 蓝双档强化池，货币经济链，8 天路线保底规则。
