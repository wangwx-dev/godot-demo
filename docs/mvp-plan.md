## 执行约定

- **顺序即依赖**：M1 的敌人是 M2 刷怪的弹药，M4 的经济是 M6 休整服务的燃料——不跳步
- **每个里程碑一次试玩**：回答验收问题，顺手校准该层数值（.tres 改数不动代码）；答不上来就停下调，不带病推进
- 每里程碑一个 commit 系列，完成时更新本表状态（表格加 ✅ 前缀）
- M2 起每次试玩顺带盯性能角标：同屏 50 只 + 分离力在 GL Compatibility 下的帧率——超预算优先降分离力计算频率
- 素材：进入 Mx 前把该里程碑涉及的 asset-list 行推到 🟨（占位不欠账）
